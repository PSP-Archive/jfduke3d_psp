Introduction
------------
Group File Studio is a Windows utility that allows the user to browse and extract files from the 3D Realms group file format (.grp).

Installation
------------
Group File Studio should can be installed into any directory that you choose, but *every* file that comes in the zip file should be extracted to the same directory.  Then, just double click on the program itself and it will launch.
